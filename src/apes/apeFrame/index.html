Ape Framework is an easy-to-use Web3 development tool. Users can compile, test, and interact with smart contracts all in one command line session. With our modular plugin system, Ape supports multiple contract languages and chains.

INSTALL APE VIA PIP

via pipx or pip

pipx install eth-ape
pip install eth-ape

Install Ape Via Docker

docker command

docker run \ --volume $HOME/.ape:/home/harambe/.ape \ --volume $HOME/.vvm:/home/harambe/.vvm \ --volume $HOME/.solcx:/home/harambe/.solcx \ --volume $PWD:/home/harambe/project \ apeworx/ape compile
